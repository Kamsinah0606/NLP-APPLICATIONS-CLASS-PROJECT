# Sentiment Analysis Dashboard for Restaurant Reviews

This project presents a Sentiment Analysis Dashboard developed using
Transformer-based NLP models to analyze McDonald's restaurant reviews.

## Features
- Sentiment classification (Positive, Neutral, Negative)
- Emotion detection (Joy, Anger, Sadness, Fear)
- Interactive dashboard using Streamlit and Plotly

## Dataset
McDonald's Restaurant Reviews dataset (CSV format)

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
